CSS свойства

CSS-свойства: color - Задает цвет текста. font-size - Устанавливает размер шрифта. font-family - Определяет семейство шрифтов. background-color - Устанавливает цвет фона элемента. margin - Задает отступы вокруг элемента. padding - Задает внутренние отступы элемента. border - Определяет границу элемента. width - Устанавливает ширину элемента. height - Задает высоту элемента. text-align - Устанавливает выравнивание текста. display - Определяет тип отображения элемента. float - Управляет плавающим позиционированием. position - Задает позиционирование элемента. z-index - Определяет порядок наложения элементов. box-shadow - Создает тень вокруг элемента. text-decoration - Определяет оформление текста, такое как подчеркивание или перечеркивание. list-style-type - Устанавливает стиль маркера в списках. border-radius - Округляет углы элемента. transform - Применяет трансформации к элементу, такие как вращение и масштабирование. opacity - Устанавливает прозрачность элемента. position - Определяет позиционирование элемента. cursor - Устанавливает вид курсора при наведении на элемент. line-height - Устанавливает высоту строки. box-sizing - Определяет модель блока элемента. overflow - Управляет отображением содержимого за границами элемента. text-transform - Преобразует регистр текста, такой как uppercase или lowercase. background-image - Устанавливает фоновое изображение элемента. border-collapse - Определяет способ объединения границ в таблицах. letter-spacing - Задает расстояние между символами. outline - Создает контур вокруг элемента. max-width - Устанавливает максимальную ширину элемента. box-decoration-break - Управляет разделением элементов при переносе текста. column-count - Определяет количество столбцов в многоколоночных элементах. flex - Управляет гибким макетом элемента внутри контейнера. backface-visibility - Определяет видимость обратной стороны элемента при 3D-трансформациях.

HTML теги

<head> - Содержит метаинформацию о документе, такую как заголовок, мета-теги и подключение внешних ресурсов. <title> - Устанавливает заголовок документа, отображаемый во вкладке браузера. <meta> - Предоставляет метаданные о документе, такие как кодировка, автор и описание. <link> - Подключает внешние таблицы стилей, иконки и другие ресурсы. <script> - Используется для вставки JavaScript-кода в документ. <body> - Определяет тело документа, содержащее основное содержание. <div> - Создает контейнер для группировки элементов и стилизации с помощью CSS. <p> - Определяет абзац текста. <a> - Создает гиперссылку. <img> - Вставляет изображение. <ul> - Создает неупорядоченный список. <ol> - Создает упорядоченный список. <li> - Определяет элемент списка. <table> - Создает таблицу. <tr> - Определяет строку в таблице. <td> - Определяет ячейку в таблице. <form> - Создает форму для ввода данных. <input> - Определяет поле ввода. <textarea> - Создает многострочное текстовое поле. <button> - Создает кнопку. <select> - Создает выпадающий список. <label> - Определяет метку для элемента формы. <span> - Используется для стилизации небольших фрагментов текста. <header> - Определяет верхний колонтитул страницы. <nav> - Определяет навигационное меню. <footer> - Определяет нижний колонтитул страницы. <section> - Определяет раздел страницы. <article> - Определяет отдельную статью или контент. <aside> - Используется для боковой информации или блоков. <time> - Определяет время или дату. <abbr> - Помечает сокращения. <hr> - Добавляет горизонтальную линию. <iframe> - Встраивает другую веб-страницу. <audio> - Вставляет аудио-контент.
HTML и CSS

История HTML:

HTML был разработан в начале 1990-х годов Тимом Бернерсом-Ли в ЦЕРНе (Европейский центр по исследованию ядерных частиц) в Швейцарии. Основная цель HTML - создание гипертекстовых документов для распространения научной информации. С течением времени, HTML стал стандартом для создания веб-страниц.

История CSS:

CSS был создан Хаконом Вийумом и Берт Босом в конце 1990-х годов. Основной целью было разделение структуры и представления документа. Это позволило веб-разработчикам управлять внешним видом веб-страниц без изменения их структуры. С течением времени CSS стал более сложным и мощным инструментом для стилизации веб-страниц.
