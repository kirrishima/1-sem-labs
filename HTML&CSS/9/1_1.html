HTML5 и CSS3

HTML

HTML предоставляет разнообразные элементы для структурирования веб-контента. Вот некоторые дополнительные элементы HTML:

  • <img>: Элемент <img> используется для вставки изображений на веб-страницу. Для этого требуется атрибут src для указания источника изображения.
  • <table>: Элемент <table> используется для создания табличных данных. Он включает дочерние элементы, такие как <tr> (строки таблицы), <th> (заголовки таблицы) и <td> (ячейки данных таблицы).
  • <form>: Элемент <form> используется для создания интерактивных форм на веб-страницах, которые могут использоваться для ввода данных пользователем и отправки данных.

CSS

Вот еще несколько свойств CSS для стилизации элементов HTML:

  • background-color: Свойство background-color устанавливает цвет фона элемента, позволяя выбрать определенный цвет (например, background-color: #FFFF00; устанавливает желтый фон).
  • border: Свойство border используется для определения границы элемента. Оно включает свойства, такие как border-width, border-color и border-style.
  • padding: Свойство padding определяет пространство между содержимым и границей элемента (например, padding: 10px; добавляет 10 пикселей отступа со всех сторон).
