a:link: Этот селектор используется для стилизации ссылок, которые еще не были посещены. В коде установлено text-decoration: none; (убрано подчеркивание у ссылок) и color: red; (цвет текста ссылки красный).

:visited: Этот селектор применяется к ссылкам, которые уже были посещены пользователем. В коде установлено color: #8b0000; (цвет текста посещенных ссылок темно-красный).

a:hover: Этот селектор используется для стилизации ссылок при наведении на них курсора мыши. В коде установлено color: green; (цвет текста ссылки меняется на зеленый при наведении).

a:active: Этот селектор применяется к активным ссылкам, то есть в момент нажатия на них. В коде установлено color: rgb(184, 134, 11); (цвет текста активной ссылки золотистый).

.content: Этот классовый селектор применяется к элементам с классом “content”. В коде установлено width: 78vw; (ширина элемента равна 78% ширины окна просмотра).

aside: Установлено width: 20vw; (ширина элемента равна 20% ширины окна просмотра), position: fixed; (положение элемента на странице фиксировано), right: 0; и top: 0; (элемент размещен в верхнем правом углу страницы).
